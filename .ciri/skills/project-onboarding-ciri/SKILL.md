# Project Onboarding (CIRI)

This skill contains short, actionable lessons to help contributors get started working on the CIRI Copilot repository.

Navigation
- lessons/project-overview.md — What the project is and key files
- lessons/development-setup.md — Install, venv, uv sync, editable install, env vars
- lessons/build-packaging.md — Building wheel, PyInstaller notes, build.py
- lessons/test-lint.md — Running pytest, ruff, black, mypy
- lessons/debug-run.md — Running the CLI locally, common args, logs
- lessons/skills-toolkits.md — How to add a skill, toolkit, or subagent
- lessons/contributing-checklist.md — Pre-PR checklist
- lessons/domain-workflows.md — Add a model provider, add toolkit/subagent, /sync behavior

When to use
- You are onboarding to the codebase and need exact commands and file locations.
- You are creating a new skill or toolkit and want to follow repository conventions.

See the lesson files for concise commands and examples.
